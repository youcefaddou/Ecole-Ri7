Le langage CSS

Habiller son site web avec le CSS!

Le CSS (Cascading Style Sheets) est un langage qui permet de styliser et de mettre en forme les pages web.

Qu'est-ce que le CSS?

Le CSS est un langage utilisé pour décrire l'apparence des pages web. Avec le CSS, tu peux changer les couleurs, les polices, la disposition et bien plus encore pour rendre ton site web attrayant et unique.

Pourquoi utiliser le CSS?

Le CSS est essentiel pour séparer le contenu de la présentation. Cela signifie que tu peux modifier l'apparence de ton site sans toucher au contenu, ce qui rend la maintenance et les mises à jour beaucoup plus faciles.

Comment fonctionne le CSS?

Le CSS fonctionne en appliquant des règles aux éléments HTML. Chaque règle se compose d'un sélecteur et d'un ensemble de déclarations. Les sélecteurs ciblent des éléments spécifiques, tandis que les déclarations définissent les styles à appliquer.

Exemples de CSS basiques

1. Pour changer la couleur du texte :

p {
color: blue;
}

2. Pour modifier la taille de la police :

h1 {
font-size: 24px;
}

3. Pour ajouter une bordure à un élément :

div {
border: 2px solid black;
}
