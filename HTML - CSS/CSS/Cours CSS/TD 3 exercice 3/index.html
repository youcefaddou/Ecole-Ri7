A PROPOS DE L'ÉCOLE



Né d'une profonde volonté de proposer une alternative aux ecoles d'ingénieur en informatique "
traditionnelles", le projet d'école Ri7 a vu le jour a Aubagne en 2020 avec pour objectif principal de
faciliter l'insertion ou la reconversion professionnelle de nos étudiants en leur proposant une formation
opérationnelle et certifiante dans le métier d'avenir qu'est le développement logiciel.
